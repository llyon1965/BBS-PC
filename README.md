BBS-PC! was an old-school bulletin board released by Micro-Systems software for both the PC and Amiga.

Originally released by Micro-Systems Software, the final release was version 4.20 in 1987.

This project is a clean rebuild from examining the original executables.

The initial project rebuilds the 4.20 release and adds minor modern updates, allowing the project to be expanded to create a more modern BBS.

To start with, the project is designed to compile on MS DOS (or similar) using the Microsoft 4.0 C compiler.
